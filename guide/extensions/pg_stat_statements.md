# pg_stat_statements

**pgexporter_pg_stat_statements_most_executed_calls**

| Attribute | Description |
| :-------- | :---------- |
| server | The configured name/identifier for the PostgreSQL server. |
| database | Database containing the query. |
| query | The SQL query text. |
